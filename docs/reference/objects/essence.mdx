The `Essence` type provides information about an essence.

```graphql
type Essence implements Node {
  id: ID!
  essenceID: Int!
  name: String!
  symbol: String!
  tokenURI: URL!
  mwContract: ContractAddressEVM!
  collectedBy(
  after: Cursor
  first: Int
  before: Cursor
  last: Int
  orderBy: CollectOrder
): CollectConnection!
  createdBy: Profile!
}
```


### Fields

#### `id` ([`ID`](/reference/scalars/id))

`id` the unique identifier of the essence.


#### `essenceID` ([`Int`](/reference/scalars/int))

`essenceID` the essence token id.


#### `name` ([`String`](/reference/scalars/string))

`name` the name of the essence.


#### `symbol` ([`String`](/reference/scalars/string))

`symbol` the symbol of the essence.


#### `tokenURI` ([`URL`](/reference/scalars/url))

`tokenURI` the token URI of the essence.


#### `mwContract` ([`ContractAddressEVM`](/reference/scalars/contract-address-evm))

`mwContract` the contract address of the essence middleware.


#### `collectedBy` ([`CollectConnection`](/reference/objects/collect-connection))

`collectedBy` provides a paginated list of profiles that have collected this essence.
- ##### `after` ([`Cursor`](/reference/scalars/cursor))

`after` the cursor to retreive results after.


- ##### `first` ([`Int`](/reference/scalars/int))

`first` the maximum number of profiles to return after a cursor.


- ##### `before` ([`Cursor`](/reference/scalars/cursor))

`before` the cursor to retreive results before.


- ##### `last` ([`Int`](/reference/scalars/int))

`last` the maximum number of profiles to return before a cursor.


- ##### `orderBy` ([`CollectOrder`](/reference/inputs/collect-order))

`orderBy` the order in which results are returned.


#### `createdBy` ([`Profile`](/reference/objects/profile))

`createdBy` information about the profile that created this essence.


### Interfaces

#### [`Node`](/reference/interfaces/node) 

The `Node` interface follows relay.dev spec.
