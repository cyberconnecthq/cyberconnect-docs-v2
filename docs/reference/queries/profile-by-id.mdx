---


`profileByProfileID` provides information about a profile based on the profile's `profileID` input.

```graphql
profileByID(
  chainID: ChainID
  profileID: ProfileID!
): Profile
```


### Arguments

#### `chainID` ([`ChainID`](/reference/scalars/chain-id))




#### `profileID` ([`ProfileID`](/reference/scalars/profile-id))




### Type

#### [`Profile`](/reference/objects/profile) 

The `Profile` type provides information about a profile.
