---


`profilesByProfileIDs` provides information about the profiles based on the profiles' `profileIDs` list input.

```graphql
profilesByIDs(
  chainID: ChainID
  profileIDs: [ProfileID!]!
): [Profile]
```


### Arguments

#### `chainID` ([`ChainID`](/reference/scalars/chain-id))




#### `profileIDs` ([`ProfileID`](/reference/scalars/profile-id))




### Type

#### [`Profile`](/reference/objects/profile) 

The `Profile` type provides information about a profile.
