CyberConnect 为每个[社交连接](/concepts/connection/)提供 `namespace` （名称空间）作为属性，以识别连接的来源。您可以为每个连接只设置一个 `namespace` 值，然后用指定的 `namespace` 进行过滤。

当编写一个新的连接时，建议使用自定义的`namespace`。我们建议你使用一个可以作为你的唯一 ID 的字符串，例如你的应用程序、组织或平台的名称。如果没有给出 `namespace` 或 `namespace` 为空值，连接将被添加到默认的命名空间： `CyberConnect`。

当查询追随者、关注者、朋友列表或其中的数字时，结果可以通过 `namespace` 过滤，作为一个可选参数。如果没有给出 `namespace` 或 `namespace` 为空值，将返回所有命名空间的连接。
