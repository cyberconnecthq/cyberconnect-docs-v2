:::tip 提示

在开始本节之前，请先 [安装 GraphQL 套件](/get-started/quick-setup/) 。

:::

在本节中，我们将帮助你写一个简单的查询来读取一个 NFT 的所有权信息。

1. 初始化 GraphQL 客户端。

```jsx title="/src/queries/GetNFTOwnership.tsx"
const client = new GraphQLClient(CYBERCONNECT_ENDPOINT);
```

2. 写一个查询，以获得 NFT 的 `tokenId`，它的 `owner`（拥有者），以及拥有者的推特 ID `handle` 。

```jsx title="/src/queries/GetNFTOwnership.tsx"
export const GET_NFT_OWNERSHIP = gql`
  query ($contract: String!, $tokenId: String) {
    nftOwners(contract: $contract, tokenId: $tokenId) {
      tokenId
      owner
      twitter {
        handle
      }
    }
  }
`;
```

请务必查看 [CyberConnect API](/cyberconnect-api/overview/) 中可用字段的完整列表。
