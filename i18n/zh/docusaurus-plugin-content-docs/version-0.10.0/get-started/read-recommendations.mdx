:::tip 提示

开始这一节之前，请先 [安装 GraphQL 套件](/get-started/quick-setup/) 。

:::

在本节中，我们将帮助你写一个简单的查询，从[社交图谱](/concepts/social-graph/)中为一个用户获取推荐的[社交连接](/concepts/connection/)。

1. 初始化 GraphQL 客户端。

```jsx title="/src/queries/GetRecommendations.tsx"
const client = new GraphQLClient(CYBERCONNECT_ENDPOINT);
```

2. 写一个查询，以获得一个用户的推荐连接列表，包括他们的 `address`（地址）、 `domain`（域名）和 `recommendationReason`（推荐理由）。

```jsx title="/src/queries/GetRecommendations.tsx"
export const GET_RECOMMENDATIONS = gql`
  query ($address: String!) {
    recommendations(address: $address) {
      result
      data {
        list {
          address
          domain
          recommendationReason
        }
      }
    }
  }
`;
```

请务必查看 [CyberConnect API](/cyberconnect-api/overview/)中可用的字段的完整列表。
