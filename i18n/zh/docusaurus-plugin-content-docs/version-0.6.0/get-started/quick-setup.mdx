## 环境

请确保你已经安装了 [Node.js](https://nodejs.org/en/download/) 和 [MetaMask Chrome Extension](https://metamask.io/) 插件。

## 安装 JS SDK

我们需要通过 [CyberConnect JS SDK](/cyberconnect-sdk/connect-with-js-sdk/) 运行。你可以执行以下命令来进行安装：

```bash npm2yarn
npm install @cyberlab/cyberconnect
```

## 安装 Social Verifier

我们需要通过 [CyberConnect Social Verifier](/cyberconnect-sdk/connect-with-social-verifier/) 运行。你可以执行以下命令来安装：

```bash npm2yarn
npm install @cyberlab/social-verifier
```

## 安装 GraphQL packages

除此之外，我们需要使用 GraphQL 作为查询语言来进行数据检索。我们需要安装以下两个安装包：

```bash npm2yarn
npm install graphql graphql-request
```
