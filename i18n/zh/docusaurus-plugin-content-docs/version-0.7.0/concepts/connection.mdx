CyberConnect 协议允许用户形成在区块链网络上记录并验证的社交连接。 该协议支持用于不同目的的不同类型的连接：

import ConnectCards from "@site/src/components/ConnectCards";

<ConnectCards />

:::note 注意

**报告 (Report)** 是指一个用户标记另一个用户。 **监视 (Watch)** 表示一个用户正在观察另一个用户，但不一定形成 **关注 (Follow)** 的连接关系。

:::

开发者可以通过使用 [CyberConnect JS SDK](/cyberconnect-sdk/connect-with-js-sdk/) 和 [Connect with iOS SDK](/cyberconnect-sdk/connect-with-ios-sdk/) 来整合各种连接模块以使用户的体验变得多样化。

## 它是如何工作的？

我们以 “关注” 为例。 用户单击应用程序中的 “关注” 按钮可以调用 CyberConnect JavaScript 库中的 “连接” 功能。 如果这是用户当前的第一次连接，他们需要手动签署消息。然后 CyberConnect 库将生成并存储派生密钥，以授权将连接数据写入 IPFS 和 Arweave 存储。
