:::tip 提示

开始这一节之前， 请先 [安装 GraphQL 套件](/get-started/quick-setup/) 。

:::

在本节中，我们将帮助你写一个简单的查询去读取社交图谱中用户的 [身份](/concepts/identity/) 。

1. 初始化 GraphQL 客户端：

### 初始化 GraphQL 客户端：

```jsx title="/src/queries/GetIdentity.tsx"
const client = new GraphQLClient(CYBERCONNECT_ENDPOINT);
```

2. 编写一个查询，通过地址获得用户的 `domain`, `followerCount`, and `followingCount` 。

```jsx title="/src/queries/GetIdentity.tsx"
export const GET_IDENTITY = gql`
  query ($address: String!) {
    identity(address: $address) {
      domain
      followerCount
      followingCount
    }
  }
`;
```

请务必查看 [CyberConnect API](/cyberconnect-api/overview/) 中可用字段的完整列表。
