In this section, we'll help you write a simple query to read a user's [identity](/concepts/identity/) in the social graph.

1. Initialize the GraphQL client:

### Initialize the GraphQL Client

```jsx title="/src/queries/GetIdentity.tsx"
const client = new GraphQLClient(CYBERCONNECT_ENDPOINT);
```

2. Write a query to get the `domain`, `followerCount`, and `followingCount` of the user with address:

```jsx title="/src/queries/GetIdentity.tsx"
export const GET_IDENTITY = gql`
  query($address: String!) {
    identity(address: $address) {
      domain
      followerCount
      followingCount
    }
  }
`;
```

Make sure to check out the full list of fields available in the [CyberConnect API](/cyberconnect-api/overview/).
